# Stories and Blog Posts

Use this folder for published Broad Shoulders Initiative story or blog pages.

## Fast Publishing Pattern

1. Create a new HTML file with a short lowercase slug, such as `my-story-title.html`.
2. Use the structure of an existing published story page for navigation, metadata, story body, image, and support links.
3. Link the new page from a matching card in `../stories.html`.
4. Review the story for privacy, consent, identifying details, and safety before publishing.

## Image Notes

- Add new story images to `../assets/images/`.
- Keep alt text specific and useful, or use an empty alt attribute only when the image is purely decorative.
- Prefer compressed `.jpg`, `.jpeg`, or `.webp` images sized for web use before adding them to the repository.
