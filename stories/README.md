# Future Stories and Blog Posts

Use this folder for individual Broad Shoulders Initiative story or blog pages.

## Fast Publishing Pattern

1. Duplicate `story-template.html`.
2. Rename the copy with a short lowercase slug, such as `my-story-title.html`.
3. Replace the title, category, date placeholder, read time, meta description, image alt text, and body text.
4. Link the new page from the matching card in `../stories.html`.
5. Review the story for privacy, consent, identifying details, and safety before publishing.

## Image Notes

- Use `../assets/images/story-placeholder.svg` until a real post image is ready.
- Add future post images to `../assets/images/`.
- Keep alt text specific and useful, or use an empty alt attribute only when the image is purely decorative.
